# TATA iQ GenAI Powered Data Analytics Virtual Internship

## Task 1: Exploratory Data Analysis (EDA) and Delinquency Risk Profiling

### Project Overview

This project was completed as part of the **TATA iQ GenAI Powered Data Analytics Virtual Internship Program**. The objective was to perform Exploratory Data Analysis (EDA) on customer financial and behavioral data to identify key risk factors associated with credit delinquency.

---

## Business Objective

Geldium Finance aims to reduce credit card delinquency by leveraging AI-powered predictive analytics. This analysis focuses on identifying customer risk indicators that can support future delinquency prediction models.

## Key Risk Indicators Identified

* Payment History
* Credit Utilization
* Debt-to-Income Ratio
* Credit Score
* Employment Status
* Account Tenure

---

## Key Findings

* Unemployed customers exhibited the highest delinquency rate.
* Business credit card holders showed elevated risk levels.
* Missing values were limited and manageable using median imputation.
* The dataset is suitable for predictive modeling after preprocessing.
